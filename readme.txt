=== ScholarPress Vitaware ===
Contributors: clioweb, kalbers, stakats
Tags: education, cv, curriculum vitae,
Requires at least: 2.0.2
Tested up to: 3.2.1
Stable tag: 1.0

ScholarPress Vitaware enables you to build a CV in Wordpress using Zotero. It currently requires the user to create a WordPress page containing a shortcode of the form [scholarpress-vitaware user="<username>"] where <username> is a Zotero username.
